Tri-Colored Sprite

Tri-Colored Sprite is a set of asset that allows you to easily render the same sprites with different colors. The system works with a trio of color. Each specified color will be applied to a one of the 3 RGB channels. The final pixel color will be calculate as such:

pixelColor = (spriteR * colorA) + (spriteG * colorB) + (spriteB * colorC);


TriColored Component

The TriColored Component is a component that can be added to any GameObject with the SpriteRenderer component. The component will change the SpriteRenderer's shader and apply the Tri-Colored pattern with it's three colors.

Let's look at an example. Here we have the original image of a canon. Without the Tri-Colored pattern applied, it doesn't look so good. That's because the 3 color channels have been specifically arranged to be used with the technique. In our case the first channel (R) represents the lighting. Most of the image is black, but everywhere light would normally create shades or reflections, the value gets closer to 1. The remaining two channels (G and B) each represent the base color of a part of the canon. That way, we can easily change the color of the tubes independently from the main frame.



Now let's add a TriColored component to our sprite and set it's 3 colors to:
(R) light: grey
(G) color X: orange
(B) color Y: dark metallic blue



If we were to change the 3rd color (B) to, let's say, green, only the canon's tubes would get affected.



This rendering technique allows you to render the same sprite with many different colors. However, it should be noted that this technique only allows for a sprite to be composed of 3 additive colors. This means the Tri-Colored pattern is not helpful in every cases.


Scripting

Modifying the color values of the TriColored component is fairly straight-forward. However, if you want to modify more than one color at a time, you may use the SetColors(...) method instead of the standard properties to save a bit of CPU time.

TriColored triColored = GetComponent<TriColored>(); // Set individual properties triColored.ColorR = Color.red; triColored.ColorG = Color.white; triColored.ColorB = Color.black; // OR set them all at once triColored.SetColors(Color.red, Color.white, Color.black);
