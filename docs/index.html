What's CCC ?

CCC is a collection of useful C# scripts built for Unity.

They range from neat Property Drawers to complex mathematical algorithms and more. The best way to get a grasp of what CCC is would simply be to explore the documentation. Here's a brief overview of some of the things you'll find in CCC:

  - Persistent object management tools that help you set up a collection of global managers quickly
  - Alternative sprite coloring techniques
  - Common UI animations
  - Useful field attributes that provide fast and easy ways to adapt the inspector to your needs
  - Audio systems that solve common development pattern problems
  - Type extensions that make your code smaller and more readable
  - Various mathematical tools that help you control your numbers the way you want
  - File serialization systems that make loading and saving quite easier and faster
  - Various useful data structures
  - A list of time-saving UI classes like 'Infinite Scrollers'
  - And more...


Can I use CCC freely?

Absolutely! You can use/modify every CCC script however you want. The code base is entirely royalty-free and open source. This means giving CCC credit is appreciated but not mandatory :)


What's the story behind CCC?

The CCC library was gradually built by myself (Frederic Bessette) across several game projects. I began aggregating some of my scripts that were project-independent about a year ago. Many assets were added, modified or completely deleted across the months. Now, I feel like the library is sufficiently 'clean' to be shared. However, none of the scripts are perfects. Some are unfinished, and others violate good coding practices, so take this library with a grain of salt.
NB: Some of the scripts and systems were inspired by others shared on the internet.
